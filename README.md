# Hands-on-Machine-Learning-with-Scikit-Learn-Keras-and-TensorFlow
Bab 1 — Machine Learning Landscape

Memahami apa itu Machine Learning, jenis-jenisnya (supervised, unsupervised, reinforcement), dan berbagai contoh penggunaan di dunia nyata.

Fokus: memahami big picture + kapan menggunakan jenis algoritma tertentu.

Bab 2 — End-to-End Machine Learning Project

Mendemonstrasikan alur kerja proyek ML secara penuh: mulai dari pengambilan data, eksplorasi, preprocessing, training model, evaluasi, hingga deployment.

Fokus: workflow profesional dalam membuat model ML dari awal sampai siap dipakai.

Bab 3 — Classification

Masuk ke Supervised Learning untuk klasifikasi menggunakan dataset MNIST. Membahas confusion matrix, precision–recall, F1-score, dan ROC curve.

Fokus: memahami cara evaluasi model klasifikasi secara benar (tidak hanya accuracy).

Bab 4 — Training Models

Belajar bagaimana model dilatih menggunakan Gradient Descent. Mencakup linear regression, polynomial regression, dan regularization (Ridge, Lasso, Elastic Net).

Fokus: pondasi matematika dari proses pelatihan model.

Bab 5 — Support Vector Machines (SVM)

Mempelajari konsep margin, support vectors, kernel trick (Linear, Polynomial, RBF), dan SVM regression.

Fokus: algoritma klasifikasi kuat untuk dataset kecil–menengah.

Bab 6 — Decision Trees

Memahami pohon keputusan serta interpretabilitas model (white box model). Membahas impurity, entropy, dan hyperparameters seperti max_depth & min_samples_split.

Fokus: model yang paling mudah dipahami manusia (“jika–maka”).

Bab 7 — Ensemble Learning & Random Forest

Menggabungkan banyak model untuk hasil prediksi yang lebih kuat (bagging, pasting, boosting, Random Forest, stacking).

Fokus: konsep "wisdom of the crowd" untuk meningkatkan performa model.

Bab 8 — Dimensionality Reduction

Mengurangi jumlah fitur menggunakan PCA, Kernel PCA, dan LLE untuk mempercepat training dan menghindari overfitting akibat curse of dimensionality.

Fokus: penyederhanaan data tanpa kehilangan informasi penting.

Bab 9 — Unsupervised Learning Techniques

Mengeksplorasi clustering (K-Means, DBSCAN) dan Gaussian Mixture Models (GMM).

Fokus: menemukan struktur data tanpa menggunakan label.

Bab 10 — Introduction to Neural Networks (Keras)

Membangun jaringan saraf (ANN/MLP) dengan Keras, mengenal layer, activation function, dan flow backpropagation.

Fokus: masuk ke dunia Deep Learning.

Bab 11 — Deep Neural Networks Training

Cara melatih deep neural network secara efektif—batch normalization, dropout, optimizer (Adam, RMSProp), learning rate scheduling, dan mengatasi vanishing/exploding gradients.

Fokus: teknik untuk meningkatkan performa dan stabilitas training Deep Learning.
